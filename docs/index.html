UniAlarmee

매일 학교 홈페이지에서 과제를 확인하느라 지친 당신을 위한 한양대학교 블랙보드 사이트 공지, 과제 리스트 불러오기 어플리케이션

간편한 블랙보드 공지, 과제 불러오기

블랙보드 페이지에 접근하여 사용자가 듣고있는 코스의 공지, 과제를 가져와 리스트로 보여줍니다.

스케쥴을 효율적으로 관리하기 위한 캘린더

블랙보드 페이지에는 없는 스케쥴과 일정을 입력하기 위한 캘린더 기능을 제공했습니다.

What we do

지난 한학기 동안 저희가 UniAlarmee Project를 진행하며 배우고 구현했던 것들입니다.

블랙보드 홈페이지 정보 접근 API

블랙보드 홈페이지에서 제공하는 통신 API를 사용해 사용자가 듣는 수업 과목 코드, 컨텐츠를 이용하여 게시글에 접근하여 가져오도록 하였습니다.

깔끔한 캘린더 오픈소스

IOS 유저들에게 익숙한 UI의 깔끔한 캘린더 오픈소스를 사용하였습니다.

웹 크롤링

블랙보드 API에서 제공하지 않는 정보들을 얻기 위해 웹크롤링을 통해 원하는 정보를 얻을 수 있었습니다. ex)블랙보드 사용자 고유 id
